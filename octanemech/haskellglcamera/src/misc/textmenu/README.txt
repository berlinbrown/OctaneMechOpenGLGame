=== Readme ===

10/21/2007 - Camera functionality in Octane Mech

The following source is a simple camera demo which will in 
turn become the camera component in the full octane mech. 

=== Usage ===

 Make sure that ghc (Haskell Compiler) is installed and type the following
 command to build the project.

 Type 'make'
 
 Launch the mech camera executable, you should see a spinning cube.
 Enter 'a', 's', 'w', 'd' to change the camera position.
 
 ./mechcamera
